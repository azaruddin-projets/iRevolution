📱 iRevolution: A Data-Driven Exploration of Apple’s iPhone Impact in India

🚀 Project Overview

iRevolution is a data-driven web application that analyzes the impact of Apple’s iPhone in India using interactive visualizations and predictive insights.

This project combines data analytics, machine learning concepts, and web development to provide meaningful insights into iPhone pricing trends, market distribution, and user decision-making.

---

🎯 Key Features

- 📊 Interactive Dashboard (Tableau Integration)
- 📈 iPhone Price Trend Analysis
- 🤖 AI-based Price Prediction System
- 🎯 Smart Recommendation System (Buy / Wait / Good Deal)
- 📉 Discount Percentage Calculation
- 🧭 Dynamic Speedometer Visualization (SVG-based)
- 💻 Fully Responsive Modern UI (Apple-inspired design)

---

🧠 Machine Learning Concept Used

Although lightweight, the project incorporates:

- 📌 Rule-Based Prediction Model
  
  - Calculates discount percentage
  - Classifies buying decision:
    - 🔴 Better to Wait
    - 🟡 Good Deal
    - 🟢 Best Time to Buy

- 📌 Data Processing using Pandas
  
  - Structured dataset handling
  - Price comparison logic
  - Trend-based insights

⚙️ How It Works

1. User enters iPhone model
2. System retrieves pricing data
3. Calculates:
   - Launch Price
   - Current Price
   - Discount Percentage
   - Highest & Lowest Prices
4. Displays:
   - 📊 Price Trend Graph
   - 🎯 Smart Recommendation
   - 🧭 Speedometer Visualization

---

📌 Sample Output

- Discount: 42% → Best Time to Buy 🟢
- Discount: 25% → Good Deal 🟡
- Discount: 10% → Better to Wait 🔴
